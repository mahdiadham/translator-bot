# Translator Bot
A Telegram Translator Bot with NodeJS.

## Technologies :
<img src="https://skillicons.dev/icons?i=nodejs,typescript,mysql&perline=3" />

## Instructions
1. Clone the repository first and install the requirements packages :
   
   ```bash
   git clone https://github.com/mahdiadham/translator-bot.git
   cd translator-bot
   npm install
   ```
2. Add folder <code>logs</code> to the root directory.
3. Create an account in <a href="https://one-api.ir" target="_blank">One Service</a> and get your own <code>API</code> and <code>API Token</code>.
4. Create .env file in the root directory and replace the values :
   
   ```bash
   BOT_TOKEN=
   API_TOKEN=
   API_URL=
   DB_USERNAME=
   DB_PASSWORD=
   DB_HOST=
   DB_PORT=
   DB_NAME=
   ```
5. Edit the bot with command :
   
   ```bash
   npm run dev
   ```
6. Build the project :
   
   ```bash
   npm run build
   ```
7. Launch the bot with command :
   
   ```bash
   npm start
   ```
8. Enjoy the bot !
